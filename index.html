📌 K线是什么？
K线（又叫蜡烛图）是看股票一天走势最常用的图表。

🟢 红色/绿色柱子 = 一天的股价变化
⬆️ 柱子顶部 = 最高价，底部 = 最低价
⬛ 粗的部分 = 开盘价到收盘价

绿色阳线 📗：收盘价 > 开盘价（涨了）
红色阴线 📕：收盘价 < 开盘价（跌了）

💡 上影线长 = 有人在高位卖（抛压大）
💡 下影线长 = 有人在低位买（支撑强）
💡 实体大 = 当天多空一方赢了
📌 那些线是什么？
🟡 MA5 = 5日均线（过去5天平均价）
  反映短期走势，线朝上=短期涨，朝下=短期跌

🔴 MA20 = 20日均线（过去20天平均价）
  反映中期走势，也叫「生命线」

🟢 MA60 = 60日均线（过去60天平均价）
  反映长期走势，跌破这条线要小心

🔑 金叉：MA5上穿MA20 → 买入信号 📈
🔑 死叉：MA5下穿MA20 → 卖出信号 📉
📌 大佬们怎么看屏幕？
专业交易员的屏幕一般分三块：

左上 📊：K线图 + 均线（看价格）
左下 📉：成交量柱（看多少人参与）
右列 📈：RSI / MACD（看涨的势头）

他们不是看一根线决定买卖，而是多个信号共振才出手：
✅ K线站稳MA20 + RSI从30往上 + 成交量放大 → 买
❌ K线跌破MA20 + RSI从70往下 + 成交量萎缩 → 卖
